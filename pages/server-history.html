Server History

Explore the evolution of MetroBlock from its humble beginnings to the current Infinite version

MB1 MetroBlock 1

Launched in early 2022, MetroBlock 1 was the original server that started it all. Built on a simple Forge setup with basic plugins, it featured a vanilla-like experience with a few custom features like custom enchantments and a basic economy system.

View Old Screenshots Read Original Rules

MB2 MetroBlock 2

Released in mid-2022, MetroBlock 2 introduced a major overhaul with a custom plugin suite and a new world generation system. This version added the first major custom dimension - the Netherite Dimension.

[Image: MetroBlock 2 Netherite Dimension screenshot]
View Plugin List Read Patch Notes

MB3 MetroBlock 3

In late 2022, MetroBlock 3 arrived with a complete redesign of the server's core mechanics. This version introduced the first faction system, player-run towns, and a dynamic economy.

[Image: MetroBlock 3 faction system screenshot]
View Faction Rules Read Community Stories

MB4 MetroBlock 4

Released in early 2023, MetroBlock 4 was a massive technical upgrade that migrated the server from Forge to Fabric with optimized performance and reduced lag.

[Image: MetroBlock 4 redstone system screenshot]
View Tech Specs Read Developer Blog

MB5 MetroBlock 5

In mid-2023, MetroBlock 5 introduced the world of Grovis, a vast custom dimension with its own biomes, structures, and lore. This version featured a rich narrative-driven experience.

[Image: MetroBlock 5 Grovis dimension screenshot]
Read Grovis Lore View World Map

Infinite MetroBlock: Infinite

Launched in early 2026, MetroBlock: Infinite represents the pinnacle of the server's evolution. Built on a custom-modified Minecraft 1.20.1 base with an advanced plugin framework.
