Connection Guidelines

Before joining, make sure your client exactly matches the server setup. Most login failures are caused by a mismatched version, outdated files, or skipped dependencies.

  • Use the current official download set from the Downloads page
  • Keep a dedicated MetroBlock instance instead of mixing other mods into it
  • Join the Discord for downtime notices, patch alerts, and community updates

Core Rules

MetroBlock is built around long-term community play, so rules focus on protecting shared spaces, player time, and server performance.

  • Do not grief, steal, or bypass claim protections
  • Do not use exploit clients, automation that harms balance, or dupes
  • Respect event rules, staff guidance, and performance limits on large machines
  • Keep chat and public builds appropriate for a shared community space
