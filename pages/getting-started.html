Getting Started

Everything a new MetroBlock player needs to install the right version, join the server, and get established during the first session.

First-Time Setup

MetroBlock: Infinite is tuned around Minecraft 1.20.1 with Forge 47.4.20. Starting with the correct client version avoids almost every early connection issue.

Step 1

Install the correct Minecraft version

Launch the official Minecraft client and create a profile for Minecraft 1.20.1. Keep this profile separate from vanilla worlds so updates are easier to manage later.

Step 2

Install Forge 47.4.20

Use the Forge installer that matches MetroBlock: Infinite. After installation, open the launcher once to confirm the profile appears and starts cleanly.

Step 3

Download the recommended pack files

Grab the current server files from the Downloads page so your client matches the live server. This is especially important for UI, balance, and custom content.

Step 4

Join and claim your starter progress

Read the welcome prompts, complete the beginner objectives, and set a safe home before heading into longer exploration or group content.

Quick Checklist

  • Minecraft version set to 1.20.1
  • Forge version set to 47.4.20
  • MetroBlock files downloaded from the official archive
  • Server rules reviewed before first login
  • Discord joined for updates and support

What To Do First In-Game

Start by collecting basic materials, finding the spawn services area, and reading any local notices that explain the current season or server event structure.

  • Set your home in a safe location
  • Claim starter rewards before exploring too far
  • Check whether the current season has progression quests

Common Early Mistakes

Most new-player issues come from version mismatches, skipped downloads, or assuming every system works like vanilla survival.

  • Joining with the wrong loader or wrong minor version
  • Ignoring server-specific rules for claims or economy
  • Skipping the guide resources linked from this wiki
